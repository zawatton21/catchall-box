#+title: catchall-box — UUID-based file management for Org mode
#+author: Fujisawa Electric Management Office

#+begin_quote
Store files with UUID names, reference them with human-readable Org links.
#+end_quote

* What is this?

A "catchall box" is a directory where files are stored with UUID-based
names.  You reference them in Org via links like:

#+begin_example
[[catchall-box:a1b2c3d4-e5f6-7890-abcd-ef1234567890.pdf][My Report.pdf]]
#+end_example

This decouples file identity (UUID, never changes) from display name
(link description, freely editable).

* Features

| Feature | Command |
|---------+---------|
| Store file → box + insert link | =catchall-box-store-file= |
| Store webpage → box as HTML | =catchall-box-store-url= |
| Batch rename files → UUIDs | =catchall-box-rename-to-uuid= |
| Revert UUIDs → original names | =catchall-box-revert-filenames= |
| Delete file at point | =catchall-box-delete-file-at-point= |
| Find orphaned files | =catchall-box-orphan-files= |
| Export subtree as folder | =catchall-box-export-subtree= |
| Import folder → headings + UUIDs | =catchall-box-import-directory= |
| Copy file with readable name | =catchall-box-copy-file-at-point= |

* Platform support

| Platform | OneDrive auto-detect |
|----------+---------------------|
| Windows  | Registry lookup     |
| WSL      | Registry via /proc  |
| macOS    | ~/Library/CloudStorage |
| Linux    | ~/OneDrive fallback |

* Requirements

- Emacs 27.1+
- org-mode (built-in)

* Installation

** async-installer

[[https://github.com/zawatton21/async-installer][async-installer]] — a small, non-blocking package manager for Emacs. Pure Elisp syntax — no macro DSL required.

#+begin_src emacs-lisp
(async-installer-github-add "https://github.com/zawatton21/catchall-box.git"
                            :branch "master"
                            :main "catchall-box.el")
#+end_src

** straight.el

#+begin_src emacs-lisp
(straight-use-package
 '(catchall-box :type git :host github :repo "zawatton21/catchall-box"))
#+end_src

** elpaca

#+begin_src emacs-lisp
(elpaca (catchall-box :host github :repo "zawatton21/catchall-box"))
#+end_src

** use-package (with straight or elpaca)

#+begin_src emacs-lisp
;; with straight
(use-package catchall-box
  :straight (catchall-box :type git :host github :repo "zawatton21/catchall-box"))

;; with elpaca
(use-package catchall-box
  :ensure (:host github :repo "zawatton21/catchall-box"))
#+end_src

** Manual

Clone this repository and add it to your =load-path=:

#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/catchall-box")
(require 'catchall-box)
#+end_src

* Quick start

#+begin_src emacs-lisp
(require 'catchall-box)

;; Optional: auto-detect OneDrive
(setq catchall-box-onedrive-auto-detect t)
(catchall-box-auto-configure)

;; Or set directory manually
(setq catchall-box-directory "~/Documents/Catchall-Box")
(catchall-box-update-link-abbrev)
#+end_src
